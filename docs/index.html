Pedantic Indic fonts

Pedantic Indic comprises three fonts: Pedantic Devanāgarī, Pedantic Malayālam, and Pedantic Telugu. They are based on other fonts, with the addition of extra glyphs and features that are useful for transcribing manuscripts. In each font, dependent vowel signs can also be rendered as independent characters.

Pedantic Devanāgarī

Pedantic Devanāgarī is based on Sanskrit2003, developed by Omkarananda Ashram. It's license is unknown. The following glyphs have been added:

  • pṛṣṭhamātrā vowel sign ◌ॎ (U+094E)
  • puṣpikā ꣸ (U+A8F8)
  • śirorekha ꣻ (U+A8FB)
  • Jaina oṃkāra sign ꣽ (U+A8FD)

    • Pedantic Malayālam

    Pedantic Malayālam is based on Rachana, developed by Swathanthra Malayalam Computing (Open Font License 1.1/GPLv3+). The following glyphs have been added:

    • chillu m ൔ (U+0D54)
    The following conjuncts have been added:
    • cśa ച്ശ
    • cśu ച്ശു
    • cśū ച്ശൂ
    • ṅma ങ്മ
    • ṅkta ങ്ക്ത

    Pedantic Telugu

    Pedantic Telugu is based on Pothana2000, by K. Desikachary (GPLv2+). The valapalagilaka glyph has been added, and it can be selected using the sequence

    ర + ◌్ + zero-width joiner (U+200D)
    For example, the sequence
    ర + ◌్ + zero-width joiner (U+200D) + త్థ
    would produce
    ర్‍త్థ

    In addition, consonants without the headstroke can be rendered. This can be done using the sequence

    [consonant] + zero-width joiner (U+200D) + ◌్
    That is to say,
    క + zero-width joiner (U+200D) + ◌్
    will output
    క‍్
    An independent vowel sign can then be added to the consonant.

    independent vowel signs

    All three fonts have the ability to display combining vowel signs as independent characters. For vowel signs that appear to the right of the consonant, this can be done by typing the sequence

    a + zero-width joiner (U+200D) + [vowel sign]
    That is to say, in order to display a long ā vowel sign in Devanāgarī, you would type
    अ + zero-width joiner (U+200D) + ा
    This would output
    अ‍ा

    This can be very useful for indicating additions or deletions in a manuscript. For example, in HTML,

    क<del>अ + zero-width joiner + ◌ा</del>
    can be rendered as
    कअ‍ा
    Similarly,
    క‍్<ins>అ + zero-width joiner + ◌ౌ </ins>
    can be rendered as
    క‍్అ‍ౌ

    For vowel signs that appear to the left of the consonant, we need to use the OpenType alternates feature. These vowel signs are stored in the aalt (Access All Alternates) feature. In CSS, this can be activated with the font-feature-settings property. For example, to make a new class called "aalt", use the following CSS:

    .aalt { font-feature-settings: "aalt" 1; }
    Then, the class can be used to select the vowel sign like so:
    <ins class="aalt">എ</ins>ട
    which would produce
    എട
    For information on how to access OpenType features in LibreOffice, have a look at the wiki.
